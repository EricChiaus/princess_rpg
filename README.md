# Princess Unicorn Adventure 🦄👸

A delightful educational web game where a princess riding a unicorn navigates through a magical maze to reach the rainbow castle! Perfect for kids learning math in a fun, engaging way.

## 🎮 Game Concept

**Princess Unicorn Adventure** combines cute cartoon aesthetics with educational gameplay. Players help a princess on her unicorn journey through a magical road filled with obstacles and treasures. Each encounter presents math challenges that must be solved to progress safely to the rainbow castle.

### 🌟 Core Features

- **Cute Cartoon Graphics**: Hand-drawn SVG characters including princess, unicorn, rainbow castle, and magical obstacles
- **Educational Math Battles**: Randomly generated addition, subtraction, and multiplication problems
- **Heart System**: Players start with 3 hearts, lose them for wrong answers, gain them from treasure boxes
- **Interactive Road**: Click-to-battle mechanics on obstacles along the path
- **Victory Celebration**: Animated coronation when reaching the castle

### 🎯 Gameplay Mechanics

1. **Journey Start**: Princess begins at the bottom-left corner with 3 hearts
2. **Road Navigation**: Follow the magical road from start to the rainbow castle
3. **Math Battles**: Click on obstacles (stones, bats, witches, monsters) to answer math questions
4. **Treasure Rewards**: Click treasure boxes for bonus hearts with correct answers
5. **Win Condition**: Reach the rainbow castle with at least 1 heart remaining
6. **Lose Condition**: Lose all 3 hearts before reaching the destination

### 🎨 Visual Design

- **Art Style**: Cute, cartoon-inspired SVG graphics
- **Color Palette**: Purple-pink-blue gradients with magical themes
- **Animations**: Sparkle effects, bounce animations, and smooth transitions
- **UI Elements**: Heart display, reset button, and responsive design

## 🛠️ Technical Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS v3
- **Build Tool**: Vite
- **Graphics**: Custom SVG components
- **Game Logic**: React hooks and state management

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/EricChiaus/princess_rpg.git
cd princess_rpg

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
# Build the project
npm run build

# Preview production build
npm run preview
```

## 🎮 How to Play

1. **Start the Game**: Open the game in your browser
2. **View Hearts**: Check your 3 hearts in the top-left corner
3. **Follow the Road**: Look for the visible path from start to castle
4. **Click Obstacles**: Click on stones, bats, witches, or monsters to battle
5. **Answer Questions**: Solve the math problem by selecting the correct answer
6. **Collect Treasures**: Click treasure boxes for bonus hearts
7. **Reach the Castle**: Navigate to the rainbow castle to win!

### Controls

- **Mouse Click**: Click on obstacles and treasures along the road
- **Reset Button**: Restart the game at any time

## 🧮 Educational Value

- **Math Skills**: Practice addition, subtraction, and multiplication
- **Problem Solving**: Quick thinking under pressure
- **Decision Making**: Strategic choices about which obstacles to battle
- **Risk Management**: Balance between gaining hearts and avoiding danger

## 🎯 Game Elements

### Characters
- **Princess on Unicorn**: Main protagonist, cute pink princess with golden horn
- **Rainbow Castle**: Final destination with rainbow flag

### Obstacles
- **Stones**: Basic obstacles, lose 1 heart for wrong answers
- **Bats**: Flying creatures, lose 1 heart for wrong answers
- **Witches**: Magic users, lose 1 heart for wrong answers
- **Monsters**: Scary creatures, lose 1 heart for wrong answers

### Treasures
- **Treasure Boxes**: Golden chests with sparkles, gain 1 heart for correct answers

## 🏆 Win/Lose Conditions

### Victory
- Reach the rainbow castle with 1+ hearts remaining
- Celebrate with coronation animation
- Option to play again

### Game Over
- Lose all 3 hearts through wrong answers
- Screen shows "Game Over" message
- Option to try again
